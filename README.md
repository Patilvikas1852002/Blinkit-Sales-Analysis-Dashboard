Blinkit Sales Analysis Dashboard
Project Overview

This project presents an interactive Power BI dashboard developed to analyze Blinkit's grocery sales data. The dashboard provides insights into sales performance, customer ratings, outlet characteristics, and product category trends.

The objective is to help stakeholders make data-driven decisions by identifying high-performing outlet types, item categories, and sales patterns.
